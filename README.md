## 🔧 About Features

Ink-Studio offers:

- **Smooth Animations:** Using GSAP and Framer Motion, the app brings designs to life with smooth transitions.

- **Fast Loading:** Built with Vite, Ink-Studio loads quickly and runs smoothly.  
- **TypeScript Safety:** The app is coded in TypeScript, which helps reduce bugs and increase reliability.

You can explore tattoo ideas without visiting a shop. Use it to preview styles and plan your next tattoo.
